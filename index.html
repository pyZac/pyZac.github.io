Subscription Automation Bot

A Telegram bot that automates the full subscription lifecycle for an online education platform — built end-to-end with JavaScript, Stripe, and MySQL.

How it works
1
Members interact with the Telegram bot to renew their subscription, check their status, or get support — no manual admin needed.
2
Stripe webhooks fire on payment events (activation, renewal, cancellation) and the bot responds automatically — updating the MySQL database in real time.
3
When a subscription expires, the bot automatically removes the member from the relevant Telegram groups.
4
All subscription data is stored in MySQL, creating a structured history for analysis and future automation.

RAG Knowledge Base (Python + LLM)

A context-aware Q&A system built from raw Telegram chat history — using Python for data processing and an LLM to structure and retrieve knowledge.

How it works
1
Parsed raw Telegram chat export files with Python to extract and filter relevant messages from thousands of conversations.
2
Used an LLM to categorize the messages, identify useful patterns, and generate structured knowledge entries — turning raw chat into organized information.
3
Built a knowledge base from the processed output and connected it to an LLM so users can ask questions and get answers grounded in real company data — not just generic model output.
4
This is a practical introduction to the full RAG pipeline: raw data → cleaning → LLM-assisted structuring → retrieval-augmented generation.

Toxicity Dataset — HerWILL Datathon 2026

Volunteer data collection and LLM-assisted classification project to build a labeled dataset for future toxicity detection model training.

How it works
1
Collected raw text data from Facebook pages using JavaScript scraping scripts, then cleaned and prepared it for annotation.
2
Used an LLM to classify each entry into toxic, partially toxic, or non-toxic — then reviewed the outputs and iteratively refined the prompts to improve classification quality.
3
The final labeled dataset is intended to be used for training toxicity detection models — contributing to responsible AI development.
